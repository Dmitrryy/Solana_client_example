# Solana client example

## Folder structure:

* src - common sources of class implementation.
* task1/task2_3 - contain task descriptions and corresponding main files.

## Build

```
mkdir build
```
```
cd build
```
```
cmake -DCMAKE_BUILD_TYPE=Release ..
```
```
cmake --build .
```
